LC-2K Pipelined Simulator

Built a cycle-accurate LC-2K pipelined CPU simulator using C and systems-level programming concepts. Implemented all pipeline stages with a state/newState model, added full hazard support including multi-stage forwarding, load-use stall detection, and branch flushing. Used LLDB for low-level debugging and applied instruction-level parallelism to validate correctness and performance.

Developed a C++ machine-learning classifier that predicts project topics from forum posts using NLP feature extraction. Achieved 95% accuracy through probability modeling, dataset tuning, and iterative evaluation. Implemented the prediction engine with binary search trees and maps to optimize vocabulary organization, improve lookup efficiency, and speed up runtime. Used Git and LLDB throughout development.
